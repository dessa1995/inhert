프로그램 사용 설명

  1. 난이도 선택: 메인 화면에서 원하는 학습 난이도를 선택하여 시작합니다.
  2. 사례 확인: 화면에 제시된 상속 사례를 꼼꼼히 읽고 상속 관계를 파악합니다.
  3. 금액 카드 생성:
    • 금액 입력창에 '1억 5000만'과 같이 직접 금액을 입력할 수 있습니다.
    • '+1천만', '+1억' 버튼을 클릭하여 금액을 쉽게 더할 수 있습니다. '지우기' 버튼으로 초기화합니다.
    • 금액 설정 후 '생성' 버튼을 누르면 아래에 금액 카드가 만들어집니다.
  4. 상속액 분배:
    • 생성된 금액 카드를 클릭하여 선택합니다. (선택된 카드는 파란색 테두리로 표시됩니다.)
    • 금액을 배분할 상속인의 회색 영역을 클릭하면, 선택된 카드가 해당 상속인에게 이동합니다.
    • 카드를 잘못 옮겼을 경우, 옮겨진 카드를 다시 클릭하여 선택한 후 상단의 카드 생성 영역을 클릭하면 원래 위치로 돌아옵니다.
  5. 결과 확인: 모든 금액 분배가 끝나면 '결과 확인' 버튼을 눌러 정답과 해설을 확인합니다.
  6. 반복 학습: '다음 문제'를 풀어보거나 '다시 풀기'로 복습할 수 있습니다. '난이도 선택으로' 버튼을 누르면 메인 화면으로 돌아갑니다.
